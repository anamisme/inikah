Kebijakan Privasi

Terakhir diperbarui: 23 Juni 2026

Aplikasi iNikah ("Aplikasi") dikembangkan oleh KUA Karangdadap, Kabupaten Pekalongan. Kami menghormati privasi pengguna dan berkomitmen melindungi data pribadi Anda.

1. Data yang Kami Kumpulkan

Aplikasi ini tidak mengumpulkan data pribadi secara langsung. Namun, layanan pihak ketiga yang digunakan dapat mengumpulkan informasi berikut:

  • Nama lengkap (saat pencarian sertifikat melalui Google Forms/Sheets)
  • Data perangkat standar (melalui Cloudflare Pages dan Google APIs)

2. Penggunaan Data

Data yang dikumpulkan hanya digunakan untuk:

  • Menyediakan layanan pencarian sertifikat kelulusan Bimwin
  • Menampilkan informasi jadwal dan layanan KUA
  • Mengirimkan notifikasi terkait layanan pernikahan

3. Penyimpanan Lokal

Aplikasi menyimpan preferensi tema (terang/gelap) di penyimpanan lokal perangkat Anda (localStorage). Data ini tidak dikirim ke server manapun.

4. Layanan Pihak Ketiga

Aplikasi menggunakan layanan pihak ketiga berikut:

  • Google Sheets/Apps Script — untuk database sertifikat dan notifikasi
  • Google Drive — untuk penyimpanan materi bimbingan
  • Google Forms — untuk evaluasi pre-test dan post-test
  • Cloudflare Pages — untuk hosting website

Masing-masing layanan memiliki kebijakan privasi tersendiri.

5. Keamanan

Kami menggunakan koneksi HTTPS untuk melindungi data yang ditransmisikan. Namun, tidak ada metode transmisi data melalui internet yang 100% aman.

6. Hak Pengguna

Anda berhak untuk:

  • Tidak menggunakan fitur yang memerlukan input data pribadi
  • Menghapus data localStorage kapan saja melalui pengaturan browser
  • Menghubungi kami untuk pertanyaan terkait privasi

7. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, hubungi kami melalui:

  • WhatsApp: 085226466464
  • Alamat: KUA Karangdadap, Kabupaten Pekalongan

8. Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan akan ditampilkan di halaman ini dengan tanggal pembaruan terbaru.
